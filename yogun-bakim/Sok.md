## TANIM

### Şok Nedir?

**Şok:** Oksijen dağılımında ve dokuların metabolik ihtiyaçlarının karşılanmasında yetersizlik durumudur.

**Temel Özellikler:**
- Ani gelişen enerji yetmezliği
- ATP üretiminde azalma
- Hayati organların işlevlerinde bozulma
- Doku perfüzyonunda belirgin azalma
- Dokulara oksijen sunumu azalır
- Anaerobik solunum ortaya çıkar ve aşırı laktat üretimi

**Akut, ilerleyici, hayatı tehdit edici bir durumdur!**

**ERKEN TANI VE TEDAVİ şarttır!**

> **Şok ≠ Hipotansiyon** — Şok hipotansiyon olmadan da var olabilir!

## OKSİJEN DAĞILIMINI BELİRLEYEN FAKTÖRLER

### Oksijen Dağılımı (DO2) Formülü

```
Oksijen Dağılımı (DO2) = Kardiyak Output (CO) x CaO2

Kardiyak Output (CO) = Atım Hacmi x Kalp Hızı

Atım Hacmi → Ön yük, Kontraktilite, Ard yük

CaO2 (Arteryel Oksijen İçeriği) = (Hb x SaO2 x 1,34) + (0,003 x PaO2)
```

## KOMPANZE VE DEKOMPANZE ŞOK

### Kompanzasyon Mekanizması

Kompanze şokta vücudun kompansatuvar mekanizmaları devreye girer:
- **Vasküler direnç artar** (SVR artışı)
- **Kalp debisi** düşmeye başlar
- **Kan basıncı** başlangıçta korunur, ardından düşer

Kompanze şokta kan basıncı korunurken, dekompanze şokta tüm parametreler hızla bozulur.

---

### Kompanze Şok Kliniği (ATLANMAMALI!)

| Kompansatuvar Mekanizma | Klinik Bulgu |
|------------------------|--------------|
| **Kalp hızında artış** | Taşikardi |
| **SVR artışı** | Soğuk, soluk, terli cilt |
| | Uzamış kapiller geri dolum zamanı |
| | Periferik nabız zayıf, nabız basıncında daralma |
| **Splanknik vasküler dirençte artış** | Oligüri |
| | Kusma, ileus |

**Kompanze şok müdahale edilmezse → Dekompanzasyona ilerler!**

---

### Dekompanse Şok (Hipotansif)

- Mental durum değişikliği
- Taşipne, bradipne, apne
- Taşikardi, bradikardi
- Metabolik asidoz, laktik asidoz
- Kapiller dolum zamanının uzaması
- Zayıf santral nabızlar
- Oligüri-anüri

## HİPOVOLEMİK ŞOK

### Tanım

**En sık şok nedeni!**

### Nedenleri

- Gastroenterit
- Kanama (hemorajik şok) (>%25 kan kaybı)
- Yetersiz sıvı alımı
- Osmotik diürez (Diabetik ketoasidoz, ilaçlar)
- Yanık
- Üçüncü boşluklara sıvı kaybı (sepsis, anafilaksi)
- Diabetes insipidus

### Hemodinamik Profil

| Ön yük | Kontraktilite | Ard yük |
|--------|---------------|---------|
| **Azalmış** | **Normal, artmış** | **Artmış** |

### Birincil Değerlendirme (ABCDE)

| Birincil Değerlendirme | Bulgular |
|------------------------|----------|
| **A** | Bilinç kapalı değilse havayolu açıktır |
| **B** | Sessiz taşipne |
| **C** | Taşikardi |
| | Kan basıncı normal veya hipotansiyon |
| | Dar nabız basıncı (kilo kaybı >%10, kan kaybı >%30) |
| | **Zayıf ve alınmayan periferal nabız** |
| | Normal veya zayıf santral nabız |
| | **Uzamış kapiller geri dolum zamanı** |
| | Deri soluk, soğuk, terli |
| | Mental durum değişikliği |
| | Oligüri |
| **D** | Mental durum değişikliği (huzursuzluk, laterji, koma) |
| **E** | Turgor ve tonus azalmış, göz küreleri ve fontanel çökük |

## KARDİYOJENİK ŞOK

### Tanım

Kardiyak output düşüklüğü ile karakterize şok tipidir.

### Nedenleri

En sık sistolik disfonksiyon (pompa yetmezliği), bazen de diastolik disfonksiyon:

- Postoperatif KVC
- İskemik kalp hastalıkları (ALCAPA, AMI)
- Ciddi ventriküler hipertrofiye yol açan hastalıklar
- Konjenital kalp hastalıkları
- Miyokardit
- Kardiyomyopatiler
- Aritmiler
- Sepsis
- İntoksikasyon
- Travma
- Bazı nöromüsküler hastalıklar

### Hemodinamik Profil

| Ön yük | Kontraktilite | Ard yük |
|--------|---------------|---------|
| **Değişken** | **Azalmış** | **Artmış** |

### Birincil Değerlendirme (ABCDE)

| Birincil Değerlendirme | Bulgular |
|------------------------|----------|
| **A** | Bilinç kapalı değilse havayolu açıktır |
| **B** | Taşipne, solunum sıkıntısı, ral |
| **C** | Taşikardi |
| | Kan basıncı N veya ↓, nabız basıncı daralmış |
| | Zayıf periferal ve santral nabız |
| | Uzamış kapiller geri dolum zamanı |
| | KKY (pulmoner ödem, hepatomegali, jugular venöz dolgunluk, gallop ritmi) |
| | Siyanoz (KKH, pulmoner ödem) |
| | Soğuk, soluk ve terli cilt |
| | Oligüri |
| **D** | Mental durum değişikliği |
| **E** | Vücut ısı değişikliği |

## DİSTRİBÜTİF ŞOK

### Tanım

Kan volümünün uygunsuz dağılımı ile oluşur.

**Mekanizma:** Vasodilatasyon → Kapiller geçirgenlikte artış → Rölatif hipovolemi

### Alt Tipleri

1. Anaflaktik şok
2. Nörojenik şok
3. Septik şok

### Hemodinamik Özellikler

- **Kalp debisi:** Değişken
- **Sıcak şok** (hiperdinamik): Kardiyak debi ↑, SVR ↓
- **Soğuk şok** (hipodinamik): Kardiyak debi ↓, SVR ↑

### Birincil Değerlendirme (ABCDE)

| Birincil Değerlendirme | Bulgular |
|------------------------|----------|
| **A** | Bilinç kapalı değilse havayolu açıktır |
| **B** | **Taşipne**, ARDS, pnömoni ve solunum sıkıntısı |
| **C** | Taşikardi |
| | Hipotansiyon, nabız basıncı geniş veya daralmış |
| | Dolgun periferal nabız, geç dönemde zayıf |
| | Zayıf santral nabız |
| | **Hızlı** veya uzamış kapiller geri dolum zamanı |
| | **Başta sıcak**, geç döneme soğuk deri |
| | Oligüri |
| **D** | Mental durum değişikliği |
| **E** | **Vücut ısı değişikliği**, döküntü |

---

### Anaflaktik Şok

**Patofizyoloji:**
- Alerjenle karşılaşmadan saniyeler sonra gelişir
- Veno-vazodilatasyon, kapiller geçirgenlikte artış
- Pulmoner vazokonstriksiyon → Sağ kalp iş yükü artar → Kalbe venöz dönüş azalır
- Kalp debisi düşer

**Klinik Bulguları:**
- Ajitasyon, anksiyete
- Bulantı, kusma
- Ürtiker
- Anjioödem
- Stridor veya hışıltı
- Hipotansiyon
- Taşikardi

### Anaflaktik Şokta Tedavi

**İlk Basamak:**
- Havayolu, pozisyon, oksijen (%100), krikotirotomi gerekebilir
- **Adrenalin 0.01 mg/kg IM** (max. 0.3-0.5 mg), 3-5 dk tekrar, uyluk ön lateral bölge (intra müsküler)
- 20 mL/kg SF yüklemesi

**İkincil Tedavi:**
- H1 bloker: Difenhidramin 1-2 mg/kg (max. 50 mg)
- H2 bloker: Ranitidin 1-2 mg/kg (max. 50 mg)
- Steroid: Metilprednisolon 2 mg/kg İV
- Bronkospazm varsa salbutamol (2.5 mg) nebülizasyonu

**Refrakter Şokta:**
- Adrenalin infüzyonu (0.1-1 mcg/kg/dk)
- Dopamin infüzyonu (5-15 mcg/kg/dk)

---

### Nörojenik Şok

- Kalp ve damarlarda sempatik innervasyon kaybı, anormal vazodilatasyon
- Sıklıkla kafa travması, servikal travma veya toraks travmalarında
- **Geniş nabız basıncı** ile hipotansiyon
- **Normal kalp hızı** veya bradikardi

> **Kompansatuvar taşikardi yoktur!**

## SEPTİK ŞOK

> Tanınması en zor şok.
### Tanım

**SIRS + Enfeksiyon + KVS bozukluk**

### Sepsis

- Şiddetli enfeksiyona bağlı bir klinik sendromdur
- Sepsis tanımı için enfeksiyonun kanıtlanma şartı yok
- Klinik olarak yüksek ihtimalle sepsis olma olasılığı varsa, SİSTEMİK İNFLAMATUAR YANIT SENDROMU (SIRS) kriterlerini de karşılıyorsa **sepsis kabul edilir**

### Sistemik İnflamatuvar Yanıt Sendromu (SIRS) Kriterleri

Aşağıdakilerden en az **ikisi** olmalı (biri ısı veya lökosit olmalı):

1. **Taşikardi veya infantta bradikardi**
   - Kalp atım hızı yaşa göre >2 SD
   - Eksternal uyarı, ağrı, kronik ilaç alımı olmamalı
   - Veya 0.5-4 saat süresince başka şekilde açıklanamayan taşikardi
   - Veya infantta bradikardi

2. **Taşipne veya mekanik ventilasyon ihtiyacı**
   - Solunum hızının yaşa göre >2 SD üzeri olması
   - Veya ani gelişen mekanik ventilasyon endikasyonu

3. **Santral ısının >38.5°C veya <36°C**

4. **Yaşa göre lökopeni veya lökositoz veya immatür nötrofil >%10**

> **iki tanesi varsa en az**
---

### Kardiyovasküler Disfonksiyon (Septik Şok Tanımı)

1 saatte 40 mL/kg izotonik sıvı uygulanmasına rağmen:

- **Hipotansiyon:** Yaşa göre <5p veya <2 SD olması

**veya**

- **Vazoaktif ilaç kullanma gereksinimi** (epinefrin veya norepinefrin)

**veya**

- **Aşağıdaki bulgularda en az ikisinin varlığı:**
  - Açıklanamayan metabolik asidoz, baz açığı >5 mEq/L
  - Arteryal laktat düzeyinin normalin 2 katından fazla artışı
  - Oligüri (İdrar çıkışının <0.5 mL/kg/saat)
  - Uzamış kapiller geri dolum zamanı >2 sn
  - Santral ve periferal ısı farkının >3°C

## TEDAVİ

### Temel Prensipler

**Amaç:** Şokun temel sebebi ve etkilerini düzeltmek
- Perfüzyonun düzeltilmesi
- Dokulara oksijen sunumunun arttırılması

### Amaca Yönelik Tedavi

**1. Kan oksijen içeriği ve taşıma kapasitesinin artırılması:**
- %100 oksijen
- Kan transfüzyonu
- Ventilasyonun sağlanması (entübasyon)

**2. Volüm replasmanı (iki damar yolu, açılamazsa IO):**
- Sıvı tedavisi

**3. Kalp debisinin düzeltilmesi:**
- İnotropik, vazopresör, vazodilatatör

**4. Oksijen ihtiyacının azaltılması:**
- Erken entübasyon
- Sedasyon

**5. Metabolik bozuklukların düzeltilmesi:**
- Hipoglisemi, hipokalsemi, hiperkalemi, metabolik asidoz

---

### AHA 2020 Septik Şok Tedavisi

- **Sıvı:** 10-20 mL/kg
- Kristaloid ya da kolloid solüsyon
- Dengeli (RL) ya da dengesiz kristalloidler (SF)
- **Her bolus sonrası hastayı değerlendir!**
- Sıvıya dirençli septik şokta başlangıç vazoaktif ajanı olarak **epinefrin veya norepinefrin** kullanılabilir

> **adrenalin**: vazodilatasyon
> **noradrenalin**: vazokonstrüksiyon
> **dopamin**: vazokonstrüksiyon
> **dobutamin**: vazodilatasyon

---

### Surviving Sepsis Campaign - Çocuklarda İlk Resüsitasyon Algoritması

**Septik şok tanındığında ilk 1 saat içinde:**

1. IV/IO yol aç
2. Kan kültürü al
3. Ampirik geniş spektrumlu antibiyotik başla
4. Laktat ölç
5. Şok varsa sıvı bolusu uygula
6. Şok devam ederse vazoaktif ajan başla

**Devam Eden Yönetim:**
- Solunum desteği
- Pediatrik ARDS değerlendir
- Enfeksiyon kaynağı kontrolü
- Sürekli yeniden değerlendirme
- Sıvı ve vazoaktif titrasyon
- İleri hemodinamik monitorizasyon (şok devam ederse)
- Refrakter şok için hidrokortizon (+/-)
- Hipoglisemiden kaçın
- Antimikrobiyal yönetim
- VA veya VV ECLS (refrakter şok veya oksijenasyon/ventilasyon yetersizliğinde)

## TEDAVİ HEDEFLERİ

### Hedefler

- **Normal Doku Perfüzyonu**
- **Normal Doku Oksijenizasyonu**

### İyileştirilmesi Hedeflenen Klinik ve Fizyolojik Belirtiler

- **Kan basıncı:** Yaşa göre >5p sistolik kan basıncı
- **Santral ve periferik nabızların kalitesi:** Güçlü, periferik nabız santral nabıza eşittir
- **Cilt perfüzyonu:** Sıcak, KDZ <2 saniye
- **Mental durum:** Normal bilinç, GKS
- **İdrar çıkışı:** Etkili dolaşım hacminde saatte ≥1 mL/kg
