## Çocuklarda Metabolizma

- Bebek ve çocuklar büyümekte olduklarından **metabolizma hızları fazla**
- **Asit ürünler daha fazla** üretilir
- H⁺ iyonları **böbrekler yolu ile atılarak** ve vücut sıvılarında **HCO₃⁻ ile tamponlanarak** kan pH'ı sabit tutulur

***

## Kan pH'ı

| Durum | pH Değeri |
|-------|-----------|
| **Normal** | **7.35 – 7.45** |
| Yaşamla bağdaşmayan | pH <6.8 veya >7.70 |
| **Asidoz** | <7.35 |
| **Alkaloz** | >7.45 |

- Hücre içi enzim aktivitelerinin yapılması ve hücre membran stabilizasyonu için kan pH'ı **dar sınırlarda** tutulmalıdır
- **Asidoz:** H⁺ iyonu proteinlerle/enzimlerle birleşip denatürasyon → glikoz yıkımı artar → **beyin fonksiyonları bozulur**

***

## Henderson-Hasselbalch Denklemi ve Tamponlama

**pH = pK' + log (HCO₃⁻ / PaCO₂)**

- **HCO₃⁻** → Metabolik komponent
- **PaCO₂** → Respiratuar komponent

> **Görsel — Tamponlama Şeması:**
> - Metabolik hastalıklarda → HCO₃⁻'de **primer değişim**, PaCO₂'de respiratuar **kompanzasyon**
> - Respiratuar hastalıklarda → PaCO₂'de **primer değişim**, HCO₃⁻'de renal **kompanzasyon**

***

## Asit-Baz Dengesini Koruma Mekanizmaları

| Mekanizma | Açıklama |
|-----------|----------|
| **Tamponlama** | HCO₃⁻ düştükçe hücre içi tamponlar uyarılır: **Hemoglobin, Fosfat, Proteinler** |
| **Kompanzasyon** | Daha yavaş ama etkili yol |
| **Düzeltme** | Akciğerler → respiratuar komponenti; Böbrekler → metabolik komponenti düzeltir |

**Tek başına asidoz veya alkaloz nadir, genelde mikst bozukluk görülür**

***

## Arteriyel Kan Gazı Değerleri

| Bozukluk | Primer Değişiklik | Kompanzasyon |
|----------|-------------------|--------------|
| **Metabolik Asidoz** | HCO₃⁻ ↓ | PCO₂ ↓ |
| **Metabolik Alkaloz** | HCO₃⁻ ↑ | PCO₂ ↑ |
| **Respiratuar Asidoz** | PCO₂ ↑ | HCO₃⁻ ↑ |
| **Respiratuar Alkaloz** | PCO₂ ↓ | HCO₃⁻ ↓ |

## METABOLİK ASİDOZ

- Serum **HCO₃⁻ konsantrasyonunda azalma**
- **Vücut sıvılarına asit eklenmesi**

***

### Anyon Gap

Normalde hücre dışı sıvının anyon ve katyon toplamı eşittir.

$$\text{Anyon Gap} = Na^+ - (Cl^- + HCO_3^-) = 12 \pm 2 \text{ mEq/L}$$

| Değer | Anlam |
|-------|-------|
| **10-14** | Normal anyon gap |
| **>14** | Artmış anyon gap |

***

### Metabolik Asidozda Ayırıcı Tanı

**Artmış Anyon Gap — KUSMALE:**

| Harf | Neden |
|------|-------|
| **K** | Ketoasidoz (Diabetik) |
| **U** | Üremi (Böbrek yetmezliği) |
| **S** | Salisilat |
| **M** | Metanol |
| **A** | Açlık |
| **L** | Laktik asidoz |
| **E** | Etilen glikol |

| Normal Anyon Gap (Hiperkloremik) | Artmış Anyon Gap |
|----------------------------------|------------------|
| **Barsaktan HCO₃⁻ kaybı:** | **Artmış asit yapımı:** |
| - **İshal** | - Diabetik ketoasidoz |
| - Fistüller | - Laktik asidoz |
| - Üreterosigmoidotomi | - **Açlık** |
| - CaCl₂, MgCl₂ kullanımı | - Nonketotik hiperozmolar koma |
| **Renal HCO₃⁻ kaybı:** | - **Yenidoğanın metabolik hastalıkları** |
| - Karbonik anhidraz inhibitörleri | **Toksik maddeler:** |
| - **Renal tübüler asidoz** | - Salisilat |
| - Hiperparatroidizm | - Paraldehit |
| - Hipoaldosteronizm | - Metanol |
| **Diğer:** | **Asit atılımında bozukluk:** |
| - HCl uygulanması | - Akut böbrek yetmezliği |
| - Parenteral beslenme | - Kronik böbrek yetmezliği |

***

### Metabolik Asidozda Tedavi

**Sodyum bikarbonat** verilir:

$$\text{NaHCO}_3 \text{ Açığı} = (\text{İstenen pHCO}_3 - \text{Ölçülen pHCO}_3) \times \text{Vücut Ağırlığı (kg)} \times 0.3$$

- **Dağılım katsayısı:** 0.3
- **10 cc NaHCO₃ Ampül = 8.9 mEq/L NaHCO₃**

## SOLUNUMSAL ASİDOZ

- Akciğerlerdeki yetmezlik nedeni ile açığa çıkan **CO₂'nin atılamayıp plazmada artması**
- Akut ve kronik nedenlere bağlı olabilir

***

### Solunumsal Asidoz Nedenleri

| Nöromusküler Bozukluklar | Solunum Tıkanıklığı | Akciğer Hastalıkları |
|--------------------------|----------------------|----------------------|
| Beyin sistemi zedelenmesi | **Yabancı cisim** | Pnömotoraks |
| Guillain-Barré Sendromu | Aspirasyon | **Ağır pnömoni** |
| Üst kord zedelenmesi | Laringeal ödem | **Kronik akciğer hastalığı** |
| Myastenia Gravis | Ciddi bronkospazm | **Şiddetli akciğer ödemi** |
| | | **Damarsal hastalıklar** |
| | | Massif akciğer embolisi |

***

### Solunumsal Asidoz Tedavisi

- Altta yatan nedenin tedavisi
- **Yeterli solunumun sağlanması**
- O₂ verilmesi
- HCO₃⁻ verilmesi

---

## HCO3 (Bikarbonat)

> **Görsel — Proksimal Renal Tübüler Hücre Diyagramı (Fig 4):** İnterstisiyel sıvı ↔ Proksimal renal tübüler hücre ↔ Tübüler idrar. Na⁺/K⁺ ATPaz pompası, karbonik anhidraz enzimi, Na⁺/H⁺ antiporter sistemi gösterilmektedir. CO₂ + H₂O ↔ H₂CO₃ ↔ HCO₃⁻ + H⁺ reaksiyonu.

- Bikarbonatın **%85'i proksimal tübülden**, **%15'i Henle çıkan kolundan** geri emilir

**Normal HCO₃⁻ Değerleri:**

| Yaş | HCO₃⁻ |
|-----|--------|
| **<2 yaş** | 20 mEq/L |
| **Büyük çocuklar** | 22-25 mEq/L |

## METABOLİK ALKALOZ

- **Plazma pH ve HCO₃⁻ artışı**
- Respiratuar mekanizmalarla kompanze edilmeye çalışılır

***

### Renal H Sekresyonunun Arttığı Durumlar

| Durum | Mekanizma |
|-------|-----------|
| **Potasyumun düşmesi** | H⁺ iyonları hücre içine girer → HDA'ta geçici HCO₃⁻ fazlalığı |
| **Mineralokortikoid fazlalığı** | Distal nefron H⁺ atılımını uyarır |
| **Diüretik tedavi** | — |
| **Cl⁻ azlığı** | H⁺ sekresyonu artar (Cl⁻ tübüler sıvının önemli anyonudur) |

***

### Metabolik Alkalozda Klinik

Özel belirti ve bulgular yoktur:
- **Kusma, mide drenajı**
- Diüretik tedavi
- **Kas krampları**
- Halsizlik
- **Tetani**
- Hiperaktif refleksler
- **Kardiak aritmiler**

***

### Metabolik Alkalozda Tanı

| İdrar Cl⁻ < 10 mEq/L | İdrar Cl⁻ > 10 mEq/L |
|------------------------|------------------------|
| **Gastrik kayıplar** | **Normal kan basıncı:** |
| Kusma vb | - Bartter Sendromu |
| **Düşük Cl⁻ alımı** | **Hipertansiyon:** |
| Diüretik tedavisi | - Cushing Sendromu |
| | - Primer aldosteronizm |
| | - Eksojen mineralokortikoidler |

***

### Metabolik Alkalozda Tedavi

- Kaybedilen **volüm yerine konur**
- **NaCl solusyonlarına KCl eklenir**
- Aşırı mineralokortikoid etkisinde → **tümör çıkarılır** veya **spironolakton** verilir

***

### Metabolik Alkalozda Özel Durumlarda Tedavi

| Tedavi | Açıklama |
|--------|----------|
| **Asetazolamid (Diamox)** | Karbonik anhidraz inhibitörü → HCO₃⁻ atılımına neden olur. Kalp yetmezliği gibi hacim artışı olan durumlarda kullanılır |
| **HCl asit** | Çok dikkatli olunmalıdır |
| **Simetidin** | — |
| **Hemodiyaliz** | — |
| **PaCO₂ artırılması** | Ventilatördeki hastalarda ventilasyon azaltılır |

## SOLUNUMSAL ALKALOZ

- CO₂ basıncında **primer düşme** nedeni ile oluşur
- **Hiperventilasyon** vardır

***

### Solunumsal Alkaloz Nedenleri

| Solunum Merkez Uyarılması | Solunumun Periferik Uyarılması | Diğer |
|---------------------------|-------------------------------|-------|
| Anksiete | Pulmoner emboli | Hepatik yetmezlik |
| **Kafa travması** | **Konjestif kalp yetmezliği** | **Gr (−) sepsis** |
| Beyin tümörleri | Pnömoni | Mekanik veya bilinçli hiperventilasyon |
| Salisilat | | |
| **Ateş** | | |
| **Ağrı** | | |
| Gebelik | | |

***

### Solunumsal Alkaloz Tedavisi

- Hiperventilasyon nedeninin düzeltilmesi
- **Ağır hipokseminin düzeltilmesi**
- Mekanik ventilasyonda ventilasyonun **azaltılması**
- O₂ ile birlikte **%3 CO₂** verilebilir

## APGAR Skorlaması

**Virginia Apgar** (1953) tarafından geliştirilmiştir. Yenidoğanın doğumdan hemen sonraki durumunu hızlıca değerlendirmek için kullanılır. **1. ve 5. dakikada** yapılır. Skor düşükse 10., 15. ve 20. dakikada da tekrarlanır.

**APGAR = Appearance, Pulse, Grimace, Activity, Respiration**

| Parametre | 0 Puan | 1 Puan | 2 Puan |
|-----------|--------|--------|--------|
| **A — Appearance (Cilt rengi)** | Soluk / Siyanotik (tüm vücut) | Gövde pembe, ekstremiteler siyanotik (akrosiyanoz) | Tüm vücut pembe |
| **P — Pulse (Kalp hızı)** | Yok (0) | <100/dk | ≥100/dk |
| **G — Grimace (Refleks yanıt)** | Yanıt yok | Yüzünü buruşturma, hafif tepki | Öksürme, hapşırma, ağlama |
| **A — Activity (Kas tonusu)** | Flask (gevşek) | Hafif fleksiyon | Aktif hareket, iyi fleksiyon |
| **R — Respiration (Solunum)** | Yok | Yavaş, düzensiz, zayıf ağlama | Güçlü ağlama, düzenli solunum |

**Toplam Skor: 0-10 arası**

| Skor | Değerlendirme |
|------|---------------|
| **7-10** | Normal (müdahale gerekmez) |
| **4-6** | Orta derecede depresyon (uyarı + O₂ desteği gerekebilir) |
| **0-3** | Ağır depresyon (acil resüsitasyon gerekir) |

**Önemli notlar:**
- **1. dakika skoru:** Doğum anındaki durumu gösterir, acil müdahale kararı için kullanılır
- **5. dakika skoru:** Resüsitasyona yanıtı ve prognozu değerlendirir; **uzun dönem nörolojik sonuçlarla** daha ilişkilidir
- 5\. dakikada skor hâlâ <7 ise → 5 dakika arayla 20. dakikaya kadar tekrarlanır
- APGAR skoru tek başına **asfiksi tanısı koymak için yeterli değildir** (kord pH'ı, organ hasarı bulguları da gerekir)
